<label> Defines a label for an element such as:
- <button>
- <input>
- <meter>
- <output>
- <progress>
- <select>
- <textarea>
When the user clicks on the text within the <label> element, it toggles the control. The for attribute of the <label>tag should be equal to the id attribute of the related element to bind them together. 
<label for="element-id"></label>
